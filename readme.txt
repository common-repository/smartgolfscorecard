== Description ==

This plugin enables your site to displays top 100 rounds for a given golf course. This data comes from smartgolfscorecard.com



== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `smartgolfscorecard.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place [smartgolfscorecard course_id=] with a valid course_id in any content area

== Frequently Asked Questions ==

= Where do I get the course_id from? =

To get your golf course id you will need to navigate to you course profile on smartgolfscorecard.com. You can do so by 
searching for your course via http://smartgolfscorecard.com/courses/search/ and then click on the course name link. In the
address bar you will see the following "http://smartgolfscorecard.com/club/34/" the last part (number) of the URL is the course_id.
